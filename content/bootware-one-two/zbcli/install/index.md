## zbcli install

### Description

Requires superuser privilege.  

Installs the Bootware sofware. 

### Usage

```
zbcli install [OPTIONS]

Options:
  -y, --yes   Says `yes` to all prompts. This includes rebooting your system
  -h, --help  Print help
```

### Example

```
sudo zbcli install
```
