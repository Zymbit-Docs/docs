## zbcli rollback-swap

### Description

Requires superuser privilege.

Performs rollback swap between active and non-active root partitions.

### Usage

```
zbcli rollback-swap

Options:
  -h, --help  Print help
```

### Examples

```
sudo zbcli rollback-swap
```
